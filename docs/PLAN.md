# План разработки Barkhan Animated Site

## Этап 1. Основа проекта
- React + TypeScript + Vite
- базовая структура страниц
- подготовка Railway-конфигурации

## Этап 2. Материалы
- логотип
- фотографии бургера и ингредиентов
- коробка и упаковка
- короткие MP4/WebM-сцены

## Этап 3. Главная анимация
- прокрутка вниз собирает бургер по слоям
- прокрутка вверх разбирает его обратно
- после сборки бургер перемещается в коробку
- коробка закрывается
- появляется кнопка «Заказать»

## Этап 4. Остальные блоки
- меню
- преимущества
- адрес и контакты
- ссылки на заказ и социальные сети

## Этап 5. Качество
- мобильная адаптация
- оптимизация изображений и видео
- плавность анимации
- reduced-motion режим
- проверка скорости загрузки

## Этап 6. Railway
- подключение GitHub-репозитория
- команда сборки: `npm run build`
- команда запуска: `npm run start`
- автоматический deploy после push в `main`

## Следующий шаг
Создать в репозитории рабочую основу React/Vite, которую Railway сможет собрать и запустить.
